# コードスタイルルール

## バナーコメント禁止

`// ====` や `// ----` のようなセクション区切りコメントを使わない

**禁止**:
```
// ==================== 定数 ====================
// -------------------------------------------
# ============================================
<!-- ======================================= -->
```

**許可**: 通常のコメントのみ

## 全角括弧禁止

全角括弧 `（）` を使わない。半角 `()` を使用する

**禁止**: `（必須）`
**許可**: `(必須)`

## PR/Issue作成の許可

PRやIssueを作成する前に、必ずユーザーの明示的な許可を取る

## リファクタリングの定義

コメントを削除することはリファクタリングではない。リファクタリングとは外部から見た振る舞いを変えずにコードの内部構造を改善すること
